# Skill Writing Memory

这份文件作为以后写 skill 的固定约定。

## 约定

- `SKILL.md` 的规则正文、标题、步骤、机器可执行指令可以用英文。
- `SKILL.md` 里的说明性注释、补充提示、注意事项、给人看的解释统一用中文。
- 命令、路径、环境变量、文件名、代码标识符保持原技术语言，不强行翻译。

## 适用范围

- 新建自定义 skill
- 修改现有自定义 skill
- 补充 `SKILL.md` 的说明段落
- 给未来的 skill 增加注释或 caveat

## 简化记法

- 规则用英文
- `SKILL.md` 注释用中文
